## Contributing

Contributions are welcomed, just open a PR and I'll have it tested and merged. The onion URL is the only mandatory column, the rest is just nice-to-have metadata about the relay.

### Operator npubs
It is generally preferred to use NIP-19 strings in the `operator` column but feel free to use services like njump to link to the operator too.
